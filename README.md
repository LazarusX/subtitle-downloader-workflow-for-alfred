# 下载字幕 Alfred Workflow
从射手网下载与当前 Finder 窗口目录中视频文件匹配的字幕。

## 要求
* Alfred 2 with Power Pack

## 用法
1. 下载并导入`下载字幕.alfredworkflow`
1. 激活 Alfred，输入关键字`subtitlte`
